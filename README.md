## Install

```bash
docker pull lafr4nc3/alpine-texlive-ja
```

## Usage

```bash
$ docker run --rm -it -v $PWD:/workdir lafr4nc3/alpine-texlive-ja
$ latexmk -C main.tex && latexmk main.tex && latexmk -c main.tex
```
